Recent Projects

  • Research Insights Extractor

    The Research Insights Extractor is an advanced system designed to automatically extract key features from medical literature, specifically from PDF files of research papers. This system utilizes the OpenAI GPT model to extract essential features such as the conclusion, number of subjects, relative risk, length of follow-up, among others. The extracted data is presented in a structured JSON format.

  • Replicated Concurrency Control and Recovery for a Simulated Distributed Database

    This project simulated a distributed database system comprising 20 sites, incorporating multi-version concurrency control, deadlock detection, replication, and failure recovery using Python. The implementation achieved 99.99% data up-time and prevented 100% of transaction deadlocks across a range of simulated real-world scenarios.

  • e-Commerce Product Recommendations

    For this project, I spearheaded end-to-end development of a cloud-native microservice for product recommendations on an e-Commerce website. The project involved the implementation of 5 REST APIs using Python and Flask, following a test-driven development plan to reduce bugs in production. A CI/CD pipeline was configured using Github Actions for seamless updates to the cloud-native microservice. The microservice was containerized using Docker and deployed on IBM Cloud using Kubernetes.

  • Analyzing the correlation between co-morbidities and COVID-19 deaths

    This project involved the analysis and visualization of the impact of 11 pre-existing condition groups on COVID-19 outcomes. It required the pre-processing of two datasets, containing 471K and 87.3 million rows, respectively using Python & Jupyter notebook. I leveraged 2 classification methods to identify correlations and 2 predictive models to generate predictions. The results were communicated through 10 visualizations created using Tableau to help identify the role co-morbidities played in case outcomes.
