Research Insights Extractor

  • The Research Insights Extractor is an advanced system designed to automatically extract key features from medical literature, specifically from PDF files of research papers.
  • This system utilizes the OpenAI GPT model to extract essential features such as the conclusion, number of subjects, relative risk, length of follow-up, among others. The extracted data is presented in a structured JSON format.
  • The project includes a user-friendly interface, developed using React and Material-UI, to display the extracted features in an easy-to-understand manner.
  • The application uses Docker for easy deployment and distribution, and Git for version control. Users are required to have Docker and Git installed on their systems to use this project.
  • The process of feature extraction and data presentation is managed via a Python backend and a React frontend. It includes reading and preprocessing the input PDF files, making API calls to OpenAI GPT, and finally presenting the extracted features on the UI.
  • The system has room for improvements such as handling of complex PDF layouts, optimizing the speed of feature extraction, expanding the set of features extracted, and more securely handling API keys.
